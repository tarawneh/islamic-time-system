واجهة GUI عربية - الإصدار 0.6.0

الملفات:
- gui_main_ar.py
- gui/gui_config_ar.json
- requirements_gui.txt

المتطلبات:
- الكود الحالي للمشروع يجب أن يكون موجودًا ويعمل
- يفضّل أن تكون ملفات MySQL الحالية موجودة إذا أردت الحفظ والقراءة
- ثبّت PySide6:
  python -m pip install -r requirements_gui.txt

التشغيل:
  python gui_main_ar.py

ما تدعمه هذه الواجهة:
- اختيار الدولة أولًا
- ثم اختيار المدينة
- اختيار التاريخ
- الحساب الكامل
- الحفظ التلقائي في MySQL عند التفعيل
- عرض آخر تشغيل محفوظ إن كانت طبقة الاسترجاع موجودة

مهم:
- تم إعداد البنية من البداية بحيث تدعم إضافة دول أخرى لاحقًا عبر gui/gui_config_ar.json
- الواجهة عربية بالكامل
