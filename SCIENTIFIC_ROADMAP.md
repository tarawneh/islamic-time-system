Islamic Time System

Author: Dr. Rami Tarawneh

## Project Vision

The Islamic Time System aims to become one of the most scientifically rigorous open platforms for:

• Lunar crescent visibility prediction  
• Hijri calendar estimation  
• Islamic prayer time computation  
• Global crescent visibility mapping

The system combines classical astronomical calculations with modern physical visibility models and computational analysis.

# Development Roadmap

## Version 0.7.x – Physical Visibility Foundations

Focus: Establishing the physical modelling layers required for realistic crescent visibility analysis.

Completed:

• Sky brightness estimation  
• Crescent–sky contrast modelling  
• Schaefer visibility model (foundation)

Next improvements:

• Visibility score unification  
• Improved ARCV / DAZ handling  
• Model comparison framework

---

## Version 0.8 – Visibility Probability Engine

Objective: Move from deterministic visibility classification to probabilistic visibility estimation.

Planned components:

• Unified Visibility Score Engine  
• Logistic probability model for crescent visibility  
• Confidence scoring for observation likelihood  
• Multi-model consensus engine

Expected capabilities:

• Visibility probability maps  
• Model agreement indicators  
• Observation difficulty estimation

---

## Version 0.9 – Global Observation Optimization

Objective: Determine optimal global observation regions.

New algorithms:

• Best Observation Band Solver  
• Optimal observation point detection  
• Twilight window evaluation  
• Adaptive grid refinement for global maps

New outputs:

• Optimal crescent observation regions  
• Global visibility probability maps  
• Observation recommendation reports

---

## Version 1.0 – Scientific Observatory Platform

Objective: Transform the system into a full scientific crescent visibility platform.

Advanced modelling:

• Full Schaefer visibility model implementation  
• Atmospheric transparency modelling  
• Aerosol and dust influence modelling  
• Telescope-assisted visibility predictions

Integration capabilities:

• Weather data integration  
• Observational report ingestion  
• Machine learning calibration from historical observations
